## Summary

This commit is a targeted two-line fix that corrects the two BLOCK findings from iteration 1 (F-40c70d9c, F-e9aa4184). Both leader registry map keys now use underscores (`qin_shi_huang`, `genghis_khan`) matching the engine's canonical `LeaderId` values confirmed in `all-leaders.ts` and `GameInitializer.ts`.
