## Findings

### F-7c719224
- severity: warn
- file: packages/web/e2e/diplomacy.spec.ts
- line: 133
- rule: testing.md § "Layer 3: Behavioral (Browser)"
- offender: `{ timeout: 100000 }`
- message: The `waitForFunction` per-iteration timeout in `advanceTurns` is 100 000 ms, but the global per-test budget is 120 000 ms; a 10-turn advance loop (`advanceTurns(page, 10)` at line 310) could theoretically exhaust the global timeout after a single slow iteration, leaving no time for the remaining 9 turns and producing a confusing test-timeout failure instead of an assertion failure.
- suggested-fix: Cap this timeout at ≤15 000 ms (matching the pattern in other specs) and give the `diplomacy.spec.ts` test block its own `.configure({ timeout: 300_000 })` if multi-turn AI tests genuinely need more wall-clock time.
- state: open

### F-32762792
- severity: warn
- file: packages/web/e2e/ai-and-map.spec.ts (and 7 other spec files)
- line: 97
- rule: testing.md § "File Locations" / S-TESTS-L3
- offender: `async function dismissBlockingEvents(page: Page) { ... }` (identical body in 8 files)
- message: The `dismissBlockingEvents` helper is copy-pasted verbatim into 8 spec files with no shared module; if the `DISMISS_EVENT` action signature or the `blocksTurn` field name changes in the engine, all 8 copies must be updated in sync or they silently diverge.
- suggested-fix: Extract `dismissBlockingEvents` (and the analogous `dispatch` pattern) into a shared `packages/web/e2e/helpers.ts` barrel and import from there in each spec.
- state: open

### F-961149de

- rule: testing.md § "Assertion Rules"
- offender: `const REACT_INTERNAL_NOISE = ['Expected static flag was missing'];`
- message: Silencing a pageerror by string-matching on an internal React 18 fiber message is fragile — the exact message text can change across React patch releases, and if it does the filter either stops working (noise reappears) or keeps matching a different error class; the filter has no expiry/review mechanism.

### F-4b1570db

- rule: testing.md § "Layer 3: Behavioral (Browser)"
- offender: `timeout: 120000`
