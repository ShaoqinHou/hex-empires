## Findings

### F-c951a63d
- severity: warn
- file: packages/web/src/ui/panels/ReligionPanel.tsx
- line: 58
- rule: panels.md § "Styling"
- offender: `const DIVIDER: React.CSSProperties = { borderBottom: '1px solid var(--color-border)' };`
- message: Section dividers are chrome-adjacent surfaces; the rule says they must use `var(--panel-*)` tokens, but `--color-border` is a global palette token from `palette-tokens.css` — the correct panel-layer equivalent is `var(--panel-border)` (opaque) or `var(--panel-muted-border)` (30% alpha, visually softer for an interior divider).
- suggested-fix: Change to `{ borderBottom: '1px solid var(--panel-muted-border)' }` to use the panel-layer token and better reflect the "interior divider" use-case (30% alpha bronze vs 85% alpha bronze of `--panel-border`).
- state: open

### F-a52aa30b
