# Hex Empires - Quick Start Guide

## Getting Started

### Objective
Build a civilization that stands the test of time! Expand your cities, research technologies, build improvements, and compete for victory.

## Basic Controls

### Movement
- **Click unit** → Select it
- **Click blue hex** → Move selected unit
- **Space** → Cycle to next unit with movement
- **WASD / Arrows** → Pan the map
- **Scroll** → Zoom in/out

### Cities
- **Settlers** → Found new cities (click settler, then click "Found City" button or press B)
- **Builders** → Construct improvements on tiles (click builder, then click tile to build)

### Resources
- **🌾 Food** → Grows population
- **🔨 Production** → Builds units/buildings
- **💰 Gold** → Purchases items, upkeep
- **🔬 Science** → Researches technologies
- **🎭 Culture** → Researches civics
- **🙏 Faith** → Religious actions

## Game Flow

### Each Turn
1. **Move your units** - Explore, attack enemies, position for defense
2. **Build improvements** - Use builders to develop tiles (farms, mines, etc.)
3. **Manage cities** - Set production, adjust specializations
4. **Research** - Choose technologies and civics to unlock new abilities
5. **End turn** - Press Enter or click the End Turn button

### Ages
- **Antiquity Age** → Build your foundation
- **Exploration Age** → Expand and explore (requires age progress)
- **Modern Age** → Dominate with advanced technology

## Victory Conditions

- **Domination** → Conquer all enemy capitals
- **Science** → Complete all research and build spaceship parts
- **Culture** → Maximize tourism and influence
- **Score** → Have the highest score by turn 500

## Tips for New Players

1. **Expand early** - Found 2-3 cities in the first few turns
2. **Build improvements** - Farms near cities, mines on hills
3. **Balance production** - Don't neglect military units
4. **Research wisely** - Choose techs that match your strategy
5. **Monitor resources** - Avoid starving cities or gold deficits

## UI Panels

- **📚 Tech** - Technology tree
- **🎭 Civics** - Civic policy tree
- **🤝 Diplomacy** - Relations with other players
- **⚡ Ages** - Age transition and legacy bonuses
- **📊 Turn Summary** - Review your economy and warnings
