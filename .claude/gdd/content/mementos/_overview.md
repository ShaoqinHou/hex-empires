## Purpose in the game

Pre-game loadout items equipped to leader. 2 slots per leader. Earned through Legends meta-progression (Foundation Path levels 1-50; per-leader Paths levels 1-10). Cross-leader: any Memento can equip on any leader.

## Category-wide rules

- 2 Memento slots per leader (uniform across all leaders)
- Slot 1 unlocks at Foundation Path L2; Slot 2 at L5
- Equipped pre-game; locked for that game
- Cross-leader equippable once unlocked
- 4+ effect categories: Attribute grants, per-Age bonuses, free-unit-triggers, combat/unit-type bonuses
- Multiplayer host can disable Mementos per session
- No pay-to-unlock; all earned via Challenges
