## Purpose in the game

Exploration-age religion founding (NOT derived from Antiquity pantheons as in VI). Player invests Faith; founds a religion with Founder Belief + Follower Beliefs. Religion spreads via missionaries/passive. Religious victory via Cultural Legacy Path (relics).

## Category-wide rules

- Founded in Exploration age (not Antiquity)
- NOT derived from Antiquity pantheon (VI pipeline severed)
- Belief slots: Founder (1-3) + Follower (1-2) + Reliquary (for Relic slots)
- Spread mechanisms: missionaries, passive pressure, conversion on conquest
- Per-city religion tracking; dominant religion per settlement
- Toshakhana Legacy Path needs Relics which come from Reliquary Belief
- Modern age: religion persists but reduced mechanical role
