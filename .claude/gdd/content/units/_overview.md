## Purpose in the game

Military + civilian + naval + aerial units. Most non-commander units do NOT persist across age transitions. Commanders explicitly persist with XP + promotions. Civ-unique units are Ageless.

## Category-wide rules

- Categories: Melee, Ranged, Cavalry, Anti-Cavalry, Naval, Ranged Naval, Civilian, Support
- Units per age: specific roster per age
- Only Commanders persist across transitions explicitly
- Civ-unique units are Ageless (fulfill old+new civ roster simultaneously if combined)
- Units produced from Cities (not Towns) via Production queue
- Strategic resource requirements may apply (Modern armored units etc.)
