## Purpose in the game

Per-age civic trees. Each civ has a unique civic only they can research. Grants governments, policies, traditions. Tree resets at age transition.

## Category-wide rules

- Per-age civic trees (NOT cumulative)
- Each civ has a unique civic (locked to that civ)
- Unlocks: governments, policies, traditions, wonders, units, improvements
- Culture yield drives progress
- Mastery mechanic (same as tech tree)
- Ideology civics (Modern age) - Democracy/Fascism/Communism [INFERRED names]
